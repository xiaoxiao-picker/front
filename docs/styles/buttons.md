
# 自定义按钮样式的规范
=================================================================

1、按钮样式定义在文件 app/styles/global/button.scss 文件中。
---------------------------------------------------------------------------------------------------------------------

	<button class="btn btn-xx-success btn-sm"></button>

1、按钮必须引用 .btn 样式，默认为灰底。
2、自定义按钮 className 命名规则 为 btn-xx-ooooo。
3、按钮颜色值在文件头部以变量 定义。
